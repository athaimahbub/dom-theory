1.What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model.During loading a page,Document Object Model of the page is created by a browser. The object's structure and content are presented by DOM in a web page. For making a webpage dynamic,DOM gives power to JavaScript so that the elements,attributes,CSS styles etc. can be changed according to user's interaction.

2. How will you select HTML elements using DOM? Name the DOM methods.

HTML elements using DOM will be selected in six different methods.They are - 1.getElementById – This method searching element by element's id 2.getElementsByTagName –This method searching element by tag name 3.getElementsByClassName –This method searching element by class name 4.getElementsByName –This method searching element by name attribute 5.querySelector – This method returns the first element that matches the specified selector 6.querySelectorAll – This method returns elements that match the specified selector

3.What is event bubble?

Event Bubbling is a event propagation method in HTML DOM.Both elements have registered to handle a event,when an event is in an element inside another element. It's processing starts with the triggered event element and then bubbles up to the containing elements in the hierarchy. In event bubbling, the innermost element event is first captured and handled and then propagated to outer elements.

4.What is a callback function and why will you use it?

Functions are objects in JavaScript. We can pass objects to functions as parameters. We can also pass functions as parameters to other functions and call them inside the outer functions. A callback function executes code in response to an event.These events might be user-initiated, such as mouse clicks or typing. With a callback function, we can instruct application like that when the user clicks a key on the keyboard,execute this code every time. To handle asynchronous programming and avoiding problems and errors,callback function is used.
